Overview

HackTools++ ("the Extension") is a Chrome extension that provides a security testing toolkit inside Chrome DevTools, including Repeater, Intruder, Decoder, and Scanner. This privacy policy explains how the Extension handles user data.

Data Collection

The Extension does not collect, store, or transmit any personal data to external servers.

All data is processed and stored locally on your device:

  • Captured requests: Displayed in the extension panel; not sent to any server
  • Scope/Block lists: Stored in Chrome's local storage on your device
  • Saved requests: Stored locally in Chrome storage (if used)

Permissions

The Extension requests the following permissions:

PermissionPurpose
storageSave scope/block domain lists and user preferences locally
scriptingRun scanner and technology checks in the inspected page context
cookiesInclude cookies when replaying requests (user-initiated)
host_permissions: <all_urls>Replay requests to user-specified targets and capture responses in DevTools

None of these permissions are used to collect or transmit data to third parties.

Data Use

  • Requests are captured only when the user has the DevTools HackTools++ panel open
  • Replayed requests are sent directly from the extension to the target URL (user-initiated)
  • No analytics, tracking, or telemetry is used

Third-Party Sharing

The Extension does not share any data with third parties.

Security

  • All processing happens locally in your browser
  • No data is sent to external servers
  • No remote code is executed
